### Technologies:
- **Language:** PHP
- **Framework:** Laravel 6.x
- **CSS Framework:** Bootstrap 4
- **Database:** MySQL
- HTML, CSS
- **Text Editor:** Sublime Text 

### Features:
**Job Seeker:**
- View all types of jobs
- Can apply directly
- Can search job
- Can filter job category 

**Employer:**
- Can post job
- Can view applied candidate profile
